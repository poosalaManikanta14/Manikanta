8.What is diff btw local repository and remote repository

The main difference between a local repository and a remote repository is where they are located: >Local repository
A local repository is located on a user's computer. It's where a user works on their project, making modifications and committing them.
>Remote repository
A remote repository is hosted on a server that's accessible to multiple users, such as on the internet or a local network. It's where users can share their changes with their team.

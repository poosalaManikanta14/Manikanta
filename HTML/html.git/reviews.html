1.What is CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.

2.What is mean by cascading

The “cascading” in CSS refers to the fact that styling rules “cascade” down from several sources.

3.In how many ways we can insert css into html file

CSS can be added to HTML documents in 3 ways: Inline - by using the style attribute inside HTML elements. Internal - by using a 'style' element in the 'head' section.

4.Which way have more specificity and why

Inline style: Inline style has highest priority among all.
Id Selector: It has second highest priority.

5.what is meant by Box model

The box model is a fundamental concept in Cascading Style Sheets (CSS) that describes how HTML elements are laid out and sized on a web page

6.What are the values are there in box model

The CSS box model module defines the height , width , margin , and padding properties, which along with border properties make up the CSS box model.

7.Define Flex

The flex property sets the flexible length on flexible items.

8.Advantages of using flex

setting flexible width/height of elements depending on available space. both vertical and horizontal centering without any hacks and workaround solutions.

9.Flex properties

The flex CSS shorthand property sets how a flex item will grow or shrink to fit the space available in its flex container.

10.What is meant by text property

Text properties refer to the text's style settings, such as typeface, size, spacing, and alignment.

11.Define font property

It is an shorthand property to define the font-style, font-variant, font-weight, font-size/line-height, and font-family.

12.Difference between text and font

Text is the characters in a piece of writing, while a font is the style used to change the appearance of the text.

14.What is mean by HSLand HSLA

HSL stands for hue, saturation, and lightness, while HSLA is an extension of HSL that also includes an alpha channel for opacity. Both are used to define colors in Cascading Style Sheets

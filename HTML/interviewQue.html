What is HTML

HTML, or HyperText Markup Language, is the foundational language of the web, designed to structure and present content on the internet. It is a markup language, which means it uses tags to define elements within a document

what is doctype in html

The DOCTYPE declaration in HTML is an instruction to the web browser about what version of HTML the page is written in. The DOCTYPE declaration is not an HTML tag; it is a declaration that must be placed at the very top of the HTML document, before any other elements

what is the root elment of html

The element is fundamental in any HTML document, serving as the root element that encapsulates all other content within an HTML file. It's the highest-level element and sets the stage for the entire document structure, ensuring that all subsequent elements are nested within its scope.

what is tag

HTML tags are the building blocks of web pages, defining the structure and content of a webpage. They are used to create elements, which are the various parts of a webpage, such as headings, paragraphs, links, images, and more. Each HTML tag encloses a specific content and provides information about that content to the browser.

what is the d/f btw the inline vs block elements

Block elements: They consume the entire width available irrespective of their sufficiency. They always start in a new line and have top and bottom margins. It does not contain any other elements next to it. Inline elements: Inline elements occupy only enough width that is sufficient to it and allows other elements next to it which are inline. Inline elements don’t start from a new line and don’t have top and bottom margins as block elements have.

Semantic HTML elements:- These semantic elements simply mean, elements with meaning. The reason being, there definition in the code tells the browser and the developer what they are supposed to do. Framing in simpler words, these elements describe the type of content they are supposed to contain. Non-Semantic elements:- Unlike, semantic elements they don’t have any meaning. what are href vs src attributes

In HTML5, the href and src attributes play distinct roles in defining paths or URLs, each associated with specific HTML elements. The href attribute, commonly found in an anchor () elements, points to the destination of hyperlinks, facilitating navigation. The src attribute, used with elements like img and script, specifies the source URL for embedding external content like images, scripts, or multimedia.

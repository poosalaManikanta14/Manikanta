Frontend Technologies:--

HTML-(HYPERTEXT Markup Language):-

The foundation for structuring web content.

CSS-(Cascading Style Sheets):-

Controls the layout and styling of HTML elements.

JavaScript:-

Adds interactivity, animations, and logic to web pages.

Bootstrap:-

A popular CSS framework for building responsive, mobile-first designs.

Back-End Technologies:-

JavaScript(Node.js):-

Allows JavaScript to run on the server sides

Python (Django, Flask):-

Popular for readability and versatility.

Java (Spring Boot):-

Known for scalability, often used in large-scale applications.
